# UI-UX-PRO-MAX Skill（精简核心说明，有关详细技术文档/所有枚举请访问官方仓库）

**来源**：[https://github.com/nextlevelbuilder/ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill)

> UI-UX-PRO-MAX 是个人/团队型 AI 编程助手专用的顶级 UI/UX 设计智能技能集，融合 67+ UI 风格、161 色彩系统、57 字体组合、98 条 UX 准则、24 种可视化与跨 15+ 主流前端技术栈模板。支持自然语言提示自动生成设计体系、优选代码、规范 UX 并批量生成 UI 元素及组件。

## 主要能力
- 一键生成美学与体验兼优的 UI 界面方案（React、Vue、Tailwind、Svelte、SwiftUI、Flutter、HTML、Android 等）
- 智能推荐/批量组合配色/字体/动画/微交互、可一键校对可访问性与行业规范
- 支持知识检索、模板设计、代码复用、风格比对与品牌统一性适配
- 可自动迭代完善设计系统、导出 Figma/Sketch/PSD 设计稿
- 支持 prompt 式需求描述与功能提示
